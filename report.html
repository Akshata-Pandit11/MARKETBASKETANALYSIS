Market Basket Analysisis a data mining technique used to uncover patterns and relationships between products purchased together. By analyzing large datasets of customer transactions, businesses can gain valuable insights into consumer behavior. Essentially, market basket analysis identifies items frequently purchased together. For instance, if customers often buy bread and butter together, the analysis will reveal this association. For the same we use Apriori algorithm-association rule to find the insights in the dataset.

Apriori algorithm is a foundational technique in data mining, particularly in the realm of market basket analysis. It's designed to efficiently uncover frequent itemsets and generate association rules from large transactional datasets. The algorithm hinges on a straightforward principle: if an itemset  is frequent, then all its subsets must also be frequent. This is known as the Apriori property.Leveraging this property, Apriori employs a level-wise search approach to identify frequent itemsets.

Itemset is a collection of items. Support is the proportion of transactions containing a specific itemset. Confidence is the conditional probability of buying an item B, given that item A was bought.
Lift is the Measures the increase in the probability of buying item B when item A is also bought. Association Rule: A rule that indicates the relationship between two itemsets. It is represented as {antecedent} -> {consequent}.

MARKET BASKET ANALYSIS ON GROCERY DATASET

PROBLEM STATEMENT: 
To efficiently analyzing the hidden patterns and relations in the grocery transction data of a supermarket. By identifying the most frequent of most happened pattern in the purchasing to understand the buyer's behavior towards different products. The Goal of the project is to optimize the product placement,creating effective cross-selling strategies such that it will help to enhance the sales and overall performance of the supermarket.

METHODOLOGY:
Data collection and preprocessing:
Dataset we used here is sourced from the Kaggle website "https://www.kaggle.com/datasets/heeraldedhia/groceries-dataset. The attributes are:Member number i.e the unique id for the customers given bty the supermarket. Date i.e billing date or the date when item is purchased. ItemDescription contains the detail about the purchased itemset.The dataset has 38765 rows and 3 columns of object type.Data Cleaning,by handling the missing values, outliers, and inconsistencies in the dataset. For Data transforming , we use the python libraries like pandas for handling dataframes and datetime for formatting the Date attribute.

Apriori Algorithm Application
Support Threshold Definition: Determine the minimum support value to filter frequent itemsets.
Candidate Generation: Generate candidate itemsets based on frequent itemsets from the previous level.
Support Counting: Calculate the support for each candidate itemset.
Frequent Itemset Generation: Identify itemsets with support greater than or equal to the minimum support threshold.
Iterative Process: Repeat steps 2-4 until no new frequent itemsets are found.

Association Rule Generation
Association rule mining is a data mining technique used to discover relationships between variables in large datasets. In simpler terms, it helps identify items that are frequently bought together.
Confidence Threshold: Set the minimum confidence level for association rules.
Rule Generation: Generate association rules from frequent itemsets.
Rule Evaluation: Calculate confidence and lift for each rule.
Rule Selection: Select rules with confidence and lift greater than the defined thresholds.
